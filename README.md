# 🌐 Karnataka Tourism Website

A responsive tourism website highlighting the cultural heritage, natural beauty, and historic landmarks of Karnataka. Designed to inspire travelers through clean visuals, interactive UI, and informative content.

---

## 🎯 Project Objective

To create an informative and engaging tourism portal for Karnataka using **HTML, CSS, JavaScript, and Bootstrap**, focusing on:

- **User-friendly navigation**
- **Responsive design** for all screen sizes
- **Visual storytelling** using local imagery
- **Minimal, modern UI** with cultural touch

---

## ✨ Features

- ✅ Responsive layout using Bootstrap Grid
- ✅ Fixed navbar and smooth scrolling
- ✅ Hero section with background image
- ✅ Highlighted destinations with image cards
- ✅ Contact and footer sections
- ✅ Organized file structure for easy updates

---

## 🛠️ Technologies Used

| Technology  | Purpose                           |
|-------------|-----------------------------------|
| HTML5       | Page structure and content        |
| CSS3        | Custom styling and layout         |
| JavaScript  | Interactivity and animations      |
| Bootstrap 5 | Responsive layout & UI components |
